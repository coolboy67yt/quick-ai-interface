About TrtlChat

Hello! TrtlChat is an AI chat website that is purely HTML/CSS/JS, and Vercel serverless functions.

It is super simple and basic, you can use/view the code on the the github!

If you have more questions, just contact me at contact@turtldev.online.

I'll add more information to this about page soon! I am super busy with other features, updates, bugfixes, etc..

Join my discord for updates!

 

FAQ (Frequently Asked Questions)

Are my chats private?

Yes, this website simply just acts as a interface for the API.
If you're worried about privacy, the code is open source on Github and you can see exactly how your requests/data is handled.
TL;DR: Yes, it's all private.

 

I found a bug. How do I report it?

You can open a ticket on my discord or an issue on the github.
